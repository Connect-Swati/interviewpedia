How to add a new topic:
1. Go to interviewpedia\java\data\java and add db.json.js (refer test.json.js)
2. Edit interviewpedia\java\data\java\db.json.js and amend the format of que and ans
3. Edit interviewpedia\java\data\java\topics.json.js and make an entry for new topic
4. Edit interviewpedia\java\js\includes.js and make an entry in dataJS
